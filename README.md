# Simple Interest Calculator

This project is a simple interest calculator.

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100

## Description

This calculator helps users calculate the simple interest based on:

- Principal amount
- Rate of interest
- Time period

## Example

If:
Principal = 1000  
Rate = 5%  
Time = 2 years  

Simple Interest = (1000 × 5 × 2) / 100  
Simple Interest = 100
